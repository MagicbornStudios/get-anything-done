Visual Context — Vanilla HTML Demo

This page is a plain static HTML file. No React, no framework. The Visual Context registry on the right is built by scanning [data-cid] attributes in the DOM with the same framework-agnostic vc-dom module the Next.js app uses. Alt-click any landmark to copy its search hint.

A Hero Section

This section is identified with data-cid="hero-section".

Features

Feature One

Nested landmarks get a higher depth in the registry.

Feature Two

The scan is O(nodes-with-data-cid), not O(all nodes).

Call to Action

Drop vc-dom into any HTML project — including a Kaplay / Phaser canvas wrapped in a DOM overlay div — and you get the same panel.
