# GSD Upstream Documentation (Archive)

These files are the **original GSD (Get Shit Done) upstream documentation** that shipped with the fork. They describe GSD's architecture, agents, commands, CLI, and workflows — NOT GAD's.

**GAD's documentation is the site:** https://get-anything-done.vercel.app

These files are retained as:
- Prior art reference (cited on /lineage)
- Source material for future GAD feature designs
- Context for the agent-parallelization hypothesis (gad-93)

Per decision gad-92, GAD does not maintain a parallel docs/ folder. The site IS the documentation.
