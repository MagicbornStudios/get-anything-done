# Context Framework Builder/Editor Project

**Source:** Session 2026-04-16c discussion

New GAD project (separate from get-anything-done) for exploring and building
context frameworks. Associated with user's account, not the GAD framework itself.

## What it is

A dedicated workspace for designing, testing, and iterating on context frameworks
inspired by GAD and GSD. The goal is to explore what other structured context
approaches exist beyond GAD's XML planning loop — potentially visual, graph-based,
conversational, or hybrid frameworks.

## Open questions

- How to visualize context framework design (graph editor? flow chart? code?)
- Voice-first editing with VCS landmarks — how much UI is needed?
- Should frameworks be testable against eval species to measure effectiveness?
- How does this relate to the context framework catalog type (phase 42.4)?
