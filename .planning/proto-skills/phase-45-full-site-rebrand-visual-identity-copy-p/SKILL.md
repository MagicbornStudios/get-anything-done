# coordinated-site-rebrand-pass

Phase 45 shipped a 21-task rebrand atomically by sequencing structural
work (data adapters, tab shells, API routes) before visual + copy work.
The compiled order is the load-bearing piece — running visual refresh
before structure makes regressions un-attributable.

This skill captures the sequence + the canonical artifacts (VOCAB
constants file, epigraph registry, per-tab migration waves, vocabulary
audit grep) so the next rebrand doesn't relitigate the order.
